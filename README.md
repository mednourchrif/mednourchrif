## ✨ About Me

I'm an **IT Engineering Student at ESPRIT**, focused on **backend engineering**, and actively seeking **software engineering internship opportunities**.

I build **full-stack web applications**, integrate **AI-powered features**, and design systems that solve **real business problems** — from database modeling and backend logic to frontend implementation and deployment. I care about how a feature behaves after it ships, not just how it looks in a demo — that mindset comes from maintaining a **production system** (more on that below).

I'm particularly interested in **backend engineering, software architecture, and distributed systems** — and in building applications that stay maintainable as they grow.

## 🧭 Engineering Principles

- 🏗️ Build software that is **maintainable before it is clever**
- 📈 Write code that **scales with teams and users**, not just with features
- 🧱 Prefer **clean architecture** over quick fixes
- 🔍 Debug from first principles — understand *why*, not just *what*
- 🌱 Keep learning continuously, one commit at a time
